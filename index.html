Personal Project ‚Äî Rust Code Cracker

A collaborative web tool used by thousands of players designed for Rust players that allows groups to efficiently brute-force 4-digit code locks by splitting combinations across any number of teammates in real time.

Instead of every player guessing randomly, the system assigns each participant a unique subset of combinations, preventing overlap and dramatically improving efficiency. The app supports real-time progress tracking, dark mode, and language selection, and is designed to scale cleanly with group size.
